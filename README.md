# Tiago Charão Mobile Mechanic — Landing

Mobile-first landing page for a mobile (come-to-you) mechanic service in the US.
Reached via QR code on a yard sign. Single static HTML with garage-door reveal,
spinning gears, diagnostic scan beam, spark animations, and a WhatsApp / SMS /
Call CTA stack.

## Primary actions
- **Chat on WhatsApp** → opens WhatsApp chat with `+1 (470) 773-3128`
- **Text for FREE Quote** → opens SMS to `+1 (470) 773-3128`
- **Call** → dials `+1 (470) 773-3128`

## Services
Oil Change · Brakes · Diagnostics · Suspension · Alternator · Starter · Radiator

## Deploy
Static site. Drop on Vercel (no build step) — `index.html` at root.
